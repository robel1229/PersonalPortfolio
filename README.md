## 🚀 Core Features

### 🎨 Modern UI/UX Architecture
* **Glassmorphism & Grain:** A sophisticated visual style using CSS backdrop filters and SVG noise textures for a premium feel.
* **Responsive Grid:** Fully optimized for mobile, tablet, and desktop viewing using Modern Flexbox and CSS Grid.
* **Dynamic Reveal:** Integrated scroll-reveal animations to provide a smooth, interactive user experience.

### ⚙️ Engineering Focus
* **Project Documentation:** Detailed breakdowns of ECE-specific projects including circuit designs and embedded systems.
* **Interactive Skills Radar:** A visual representation of technical proficiency across multiple engineering domains.

### 🛡️ Secure & Lightweight
* **Zero-Dependency Design:** Built using vanilla HTML5, CSS3, and JavaScript for maximum speed and SEO efficiency.
* **Direct Contact Integration:** Streamlined communication portal via integrated mail protocols.
